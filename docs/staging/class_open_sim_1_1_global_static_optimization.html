OpenSim::GlobalStaticOptimization::GlobalStaticOptimization ( const std::string &  setupFilePath)
explicit

Load a solver from an XML setup file.

Note: Use print() to save solver settings to an XML file that can be subsequently loaded via this constructor.

If the model or kinematics files are provided, they are not loaded until you call solve(). If these files are not provided, you must call the setModel() and/or setKinematicsData() functions before calling solve().

◆ solve()

Solution OpenSim::GlobalStaticOptimization::solve ( ) const

Solve for muscle activity.

You must have provide a model and kinematics data before calling this function. If the property write_solution is not 'false', then the solution is also written as files to the disk.

Returns
A struct containing muscle activation, fiber length, fiber velocity, tendon force, and other control signals.
