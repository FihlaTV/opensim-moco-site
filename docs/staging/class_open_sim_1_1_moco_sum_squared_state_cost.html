OpenSim::MocoSumSquaredStateCost Class Reference

Description

Minimize the sum of squared states, integrated over the phase.

This can be used to minimize muscle activations (if those are the only states in the system), as is done in MocoInverse.

Note
This class is still under active development and should be used with caution. In the future, this class will allow you to select which states to minimize.
