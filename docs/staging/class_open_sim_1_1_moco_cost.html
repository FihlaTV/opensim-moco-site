OpenSim::MocoCost Class Referenceabstract
Available cost terms

Description

A term in the cost functional, to be minimized.

Costs depend on the phase's initial and final states and controls, and optionally on the integral of a quantity over the phase.

For developers
Every time the problem is solved, a copy of this cost is used. An individual instance of a cost is only ever used in a single problem. Therefore, there is no need to clear cache variables that you create in initializeImpl(). Also, information stored in this cost does not persist across multiple solves.
